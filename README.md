# 📧 Email Assistant Backend

An AI-powered backend service that generates and manages emails using Google's Gemini API. Built with Spring Boot, this project focuses on intelligent email composition, automation, and scalability.

## 🚀 Features

- ✨ AI-generated email content using Gemini API
- 📩 Dynamic email creation (formal, casual, professional tones)
- ⚡ RESTful APIs for email generation and management
- 🔒 Clean layered architecture (Controller, Service, Repository)
- 📦 Scalable and production-ready backend design

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Web
- Gemini API (Google AI)
- Maven

## 📌 API Endpoints

- `POST /generate` → Generate email reply using AI

## 💡 Use Cases

- Automated email drafting
- Customer support responses
- Professional communication assistance
- Productivity tools integration

## 🧠 Learning Highlights

- Integration with external AI APIs (Gemini)
- REST API design using Spring Boot
- Handling request/response flows efficiently
- Clean code and modular backend architecture
